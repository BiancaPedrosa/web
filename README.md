# Web 1 - IFSP

Repositório de projetos práticos do curso **Desenvolvimento Web 1** do Instituto Federal de São Paulo (IFSP).

**Professor:** Bianca Pedrosa  
**Período:** 2026

---

## 📁 Projetos

### 🍰 [Bolo](./Bolo)
Projeto introdutório com HTML e CSS.
- `index.html` - Estrutura HTML básica
- `style.css` - Estilização do projeto
- **Objetivo:** Aprender conceitos fundamentais de markup e design

### 📊 [Carros](./Carros)
Projeto de tabelas com dados estruturados.
- `index.html` - Tabela de catálogo de veículos
- `estilo.css` - Estilos para formatação da tabela
- **Objetivo:** Trabalhar com tabelas HTML, dados e imagens

---

## 🎯 Tecnologias

- HTML5
- CSS3
